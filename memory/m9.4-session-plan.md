### Data Model

Key new types (pseudo-Rust):

    /// Protection quality: what the permanent is protected from (CR 702.16a).
    /// Stored on KeywordAbility to replace the bare `Protection` variant.
    pub enum ProtectionQuality {
        FromColor(Color),
        FromCardType(CardType),
        FromAll,          // "protection from everything"
        FromPlayer(PlayerId),
    }

    /// Extended keyword variant for protection.
    /// KeywordAbility::ProtectionFrom(ProtectionQuality) replaces the
    /// current bare KeywordAbility::Protection variant.

    /// Tracks spells cast this turn per player (for storm count).
    /// Added to PlayerState.
    pub spells_cast_this_turn: u32,

    /// Tracks cards drawn this turn per player (for Sylvan Library).
    /// Added to PlayerState.
    pub cards_drawn_this_turn: u32,

    /// Flag: player has no maximum hand size (for Thought Vessel, Reliquary Tower).
    /// Applied via the layer system or a simple boolean on PlayerState.
    pub no_max_hand_size: bool,   // added to PlayerState

### State Changes
- `PlayerState`: add `spells_cast_this_turn: u32`, `cards_drawn_this_turn: u32`, `no_max_hand_size: bool`
- `KeywordAbility`: replace bare `Protection` with `ProtectionFrom(ProtectionQuality)` -- BUT this is a breaking enum change. Alternative: add `ProtectionFrom(ProtectionQuality)` alongside `Protection` (deprecated), keep `Protection` for backward compat until all references updated.
- `GameState`: add `loop_detection_hashes: im::OrdMap<u64, u32>` for tracking state recurrence
- `CombatState` or `GameState`: add `storm_count: u32` field (or compute from `spells_cast_this_turn`)

### New Effect Variants
- `Effect::Scry { player: PlayerTarget, count: EffectAmount }` -- CR 701.18
- `Effect::Goad { target: EffectTarget }` -- CR 701.38
- `Effect::CopySpellOnStack { source_stack_id: EffectTarget, choose_new_targets: bool }` -- CR 707.10
- `Effect::ShuffleIntoLibrary { target: EffectTarget }` -- for Darksteel Colossus (shuffle + redirect)

### New Events
- `GameEvent::SpellCopied { original_stack_id: ObjectId, copy_stack_id: ObjectId, controller: PlayerId }` -- when storm/cascade creates a copy
- `GameEvent::CascadeExiled { player: PlayerId, cards_exiled: Vec<ObjectId> }` -- cascade exile step
- `GameEvent::CascadeCast { player: PlayerId, card_id: ObjectId }` -- cascade cast step
- `GameEvent::LoopDetected { description: String }` -- mandatory loop leads to draw
- `GameEvent::Scry { player: PlayerId, count: u32 }` -- scry action performed
- `GameEvent::ProtectionPrevented { source: ObjectId, target: ObjectId, quality: String }` -- protection prevented damage/targeting

### New TriggerConditions
- `TriggerCondition::WhenYouCastSpell { cast_count_filter: Option<u32> }` -- "when you cast your first/Nth spell" (for Alela scoping)
- `TriggerCondition::WheneverCreatureTypeYouControlDealsCombatDamage { subtype_filter: SubType }` -- for Alela second trigger

### New EffectFilter Variants
- `EffectFilter::AttachedTo(ObjectId)` -- for Equipment static abilities scoping to attached creature

### Interception Sites
- `rules/mod.rs:validate_target_protection` -- extend with protection-from-color/type checks (CR 702.16b)
- `rules/combat.rs:handle_declare_blockers` -- add protection blocking check (CR 702.16f)
- `rules/sba.rs:check_aura_sbas` -- add protection illegality check for auras (CR 702.16c)
- `rules/sba.rs:check_equipment_sbas` -- add protection illegality check for equipment (CR 702.16d)
- `effects/mod.rs:execute_effect_inner` (DealDamage) -- add protection damage prevention (CR 702.16e)
- `rules/layers.rs:apply_layer_modification` (CopyOf) -- implement actual copy logic (CR 707.2)
- `rules/abilities.rs:flush_pending_triggers` -- add trigger doubling logic (CR 603.2d)
- `rules/resolution.rs` -- after spell resolves, check storm trigger and cascade trigger
- `rules/casting.rs` -- increment `spells_cast_this_turn` on successful cast
- `effects/mod.rs:draw_one_card` -- increment `cards_drawn_this_turn` on successful draw
- `rules/turn_actions.rs` -- reset `spells_cast_this_turn` and `cards_drawn_this_turn` at turn start
- `state/hash.rs` -- add new fields to deterministic hashing
