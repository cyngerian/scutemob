## Source and method

Roster is drawn from the **marker-sweep worklist** (`marker-sweep-worklist-2026-07-16.md`,
`scutemob-88`), which audited all 742 non-`Complete` markers against the **compiled registry**
on 2026-07-16 and emitted 116 cards whose stated blocker **has shipped** (chain traced
filter→effect→layer→cost). Because the sweep post-dates every PB-AC close-step, its worklist
already excludes any card the AC4/AC5/AC6 backfill authored to Complete — so the worklist *is*
the current "still-ready" set for W-PB2. The AC4/AC5/AC6 plans (`pb-plan-AC{4,5,6}.md`) and
backfill reviews (`review-pb-ac{4,6}-backfill.md`, `review-pb-ac5.md`) were consulted to confirm
attribution of the targeting / alt-cost / phase-condition cohorts.

**Blocker-proven-shipped basis (AC 4803).** Each rostered card is currently marked
`partial`/`known_wrong`/`inert` (verified 2026-07-17 by grep over the def corpus — see the
disposition column). The primitive each card needs was traced to the compiled registry by the
sweep, and the SR-34/36 lesson applies: the **authoring probe is execution, not source-tracing**
— each authored card must have an executing test path (its primitive's existing integration test
plus, where the card adds a novel combination, a focused per-card assertion) that proves the
ability registers and produces correct game state. The card-batch-reviewer verifies this per batch.

**Exclusions (kept honest per §0 recalibration — clean yield ~17%, most cards partial/blocked).**

  Tyvar Kell (haste-to-spell), Urza's Saga (token CDA), Song of Freyalise (shared chapter),

If an author's probe **falsifies** a rostered card (the primitive turns out not to reach the
whole clause), the card is authored to its truthful marker (real blocker) rather than shipped
half-done — AC 4804 permits this and it is the correct outcome, not a failure.

## Guardrails for authors (carry into every batch prompt)
- **No gated-stub effects**: `Effect::Choose`, `MayPayOrElse`, `AddManaChoice`, and the
  `AddManaAnyColor`/`Restricted`/`OfAnyColorAmount` family are gated out of Complete
  (`effect_choose_gate`, land color gate). If a card genuinely needs one, do NOT use it —
  author to a truthful marker and note the blocker.
- **Markers use helper spellings** (`Completeness::partial(...)`/`known_wrong(...)`); the marker
  count is read from `all_cards()`, never a source regex.
- **No new Effect variants** — this is a re-authoring wave. If a card needs one, mark + file it.
- **Type-line fixes flagged in the worklist** must be applied (e.g. Elvish Promenade / Boggart
  Shenanigans / Prowess-family missing the Kindred card type).
- **Self-exclusion sac / opponent-only targeting** caveats in the worklist must be honored.
- Format via `tools/check-defs-fmt.sh --fix`; a def whose formatted output overflows 100 cols
  is split by hand (SR-35).
