## Guardrails for authors (carry into every batch prompt)
- **No gated-stub effects**: `Effect::Choose`, `MayPayOrElse`, `AddManaChoice`, and the
  `AddManaAnyColor`/`Restricted`/`OfAnyColorAmount` family are gated out of Complete
  (`effect_choose_gate`, land color gate). If a card genuinely needs one, do NOT use it â€”
  author to a truthful marker and note the blocker.
- **Markers use helper spellings** (`Completeness::partial(...)`/`known_wrong(...)`); the marker
  count is read from `all_cards()`, never a source regex.
- **No new Effect variants** â€” this is a re-authoring wave. If a card needs one, mark + file it.
- **Type-line fixes flagged in the worklist** must be applied (e.g. Elvish Promenade / Boggart
  Shenanigans / Prowess-family missing the Kindred card type).
- **Self-exclusion sac / opponent-only targeting** caveats in the worklist must be honored.
- Format via `tools/check-defs-fmt.sh --fix`; a def whose formatted output overflows 100 cols
  is split by hand (SR-35).
